.. _editors-content-element:

Using the content element
=========================

The extension ships a content element :guilabel:`Matomo Opt-Out`. You find it in
the content element wizard under the :guilabel:`Special elements` tab:

.. figure:: /Images/Editor/content-element-wizard.png
   :class: with-shadow
   :alt: "Matomo Opt-Out" content element in the content element wizard

   "Matomo Opt-Out" content element in the content element wizard

Selecting it creates a new content element. There is no additional configuration
in the content element available.
